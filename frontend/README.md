# Frontend - Sistem Administrasi RT

Frontend aplikasi Sistem Administrasi RT dibuat menggunakan React dan Vite.

Frontend ini digunakan sebagai antarmuka admin untuk mengelola data penghuni, rumah, pembayaran, pengeluaran, dashboard, dan laporan.

## Tech Stack

- React
- Vite
- Bootstrap
- Axios
- Recharts
- Lucide React
- React Hot Toast

## Instalasi Singkat

```bash
npm install
npm run dev
